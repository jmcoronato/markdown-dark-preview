# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0] - 2025-10-20

### Características iniciales

- ✅ Visor de Markdown con tema oscuro al estilo GitHub
- ✅ Atajo de teclado `Ctrl+Shift+V` para abrir el preview
- ✅ **Actualización en vivo** - Los cambios se reflejan mientras escribes
- ✅ **Syntax highlighting** completo para bloques de código
  - Más de 190 lenguajes soportados
  - Tema GitHub Dark
- ✅ **Checkboxes** (task lists) renderizados correctamente
- ✅ Soporte completo de Markdown:
  - Encabezados (H1-H6)
  - Negrita, cursiva, strikethrough
  - Listas ordenadas y desordenadas
  - Enlaces e imágenes
  - Tablas
  - Blockquotes
  - Código inline y bloques de código
  - Líneas horizontales

### Tecnologías utilizadas

- `markdown-it` - Parser de Markdown
- `highlight.js` - Syntax highlighting para código
- `markdown-it-task-lists` - Plugin para checkboxes

### Colores del tema GitHub Dark

- Fondo: `#0d1117`
- Texto: `#e6edf3`
- Keywords: `#ff7b72`
- Strings: `#a5d6ff`
- Functions: `#d2a8ff`
- Variables: `#79c0ff`
- Comments: `#8b949e`
